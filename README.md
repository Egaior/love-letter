# 💌 Love Letter

Інтерактивна анімована листівка.

## Як запустити

Просто відкрий `index.html` у браузері — більше нічого не потрібно.  
Жодних фреймворків, збірників або npm. Чистий HTML + CSS + JS.

## Структура проєкту

```
love-letter/
├── index.html          # Головна сторінка
├── css/
│   └── style.css       # Всі стилі та анімації
├── js/
│   └── main.js         # Логіка відкриття листа
├── images/
│   ├── envelope.png    # Конверт (екран 1)
│   ├── background.png  # Фон листа (екран 2)
│   └── kiss.png        # Картинка в куточку
└── README.md
```

## Як розмістити на GitHub Pages

1. Залий репозиторій на GitHub
2. Зайди в **Settings → Pages**
3. Вибери гілку `main`, папку `/ (root)`
4. Натисни **Save** — через хвилину сайт буде доступний за посиланням

## Шрифти

Використовуються Google Fonts:
- **Playfair Display** — для заголовка та підпису
- **Cormorant Garamond** — для тексту листа

Потрібне інтернет-з'єднання для їх завантаження.
